# 📊 Data Professional Survey Dashboard

## 📌 Project Overview
This project presents an interactive Power BI dashboard built using survey responses from data professionals across different countries and job roles. The dashboard provides insights into salary trends, work-life balance, salary satisfaction, programming language preferences, and the challenges of entering the data industry.

The goal of this project is to transform raw survey data into meaningful insights that can help aspiring professionals and organizations better understand the current data industry landscape.

---

## 🎯 Objectives
- Analyze salary distribution across different data roles.
- Identify the most popular programming languages among data professionals.
- Evaluate work-life balance and salary satisfaction levels.
- Understand the perceived difficulty of breaking into the data field.
- Explore respondent distribution across countries.

---

## 📈 Dashboard Features
- **Total Survey Respondents**
- **Average Age of Respondents**
- **Country-wise Distribution of Survey Participants**
- **Average Salary by Job Role**
- **Favorite Programming Languages**
- **Difficulty of Breaking into Data**
- **Work-Life Balance Satisfaction**
- **Salary Satisfaction**

---

## 🛠 Tools & Technologies Used
- Power BI Desktop
- Power Query
- DAX
- Data Visualization
- CSV Dataset

---

## 🔍 Key Insights
- Data Scientists reported the highest average salaries among surveyed roles.
- Python emerged as the most preferred programming language.
- Respondents reported higher satisfaction with work-life balance than with salary.
- Many professionals considered entering the data industry moderately difficult.
- Survey participation was concentrated in a few major countries.
